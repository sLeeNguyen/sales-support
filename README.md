# Sales management website

## Requires
- Python 3.8
- Django framework (version 3.1.2)

## Runs
**Clone repository**

        git clone https://github.com/sLeeNguyen/sales-support.git
        cd sales-support
        
**Create virtualenv**

        python3 -m venv .
        source venv/bin/activate
        
**Install libraries**

        pip3 install -r requirements.txt 
        
**And run server**

        python3 manage.py runserver
